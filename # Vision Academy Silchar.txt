# Vision Academy Silchar

A modern web application for Vision Academy Silchar, a premier coaching institute in Barak Valley dedicated to competitive exam preparation.

## Overview

Vision Academy Silchar offers coaching for various competitive exams including:
- Bank PO/Clerk Exams
- SSC (CGL, CHSL)
- CAT/MAT
- CLAT
- NDA/CDS
- Railway & LIC Exams
- ADRE (Assam Direct Recruitment Examination)

## Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **UI Components**: Custom components with Radix UI primitives
- **Language**: TypeScript
- **Image Hosting**: Remote image support for picsum.photos and placehold.co

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
